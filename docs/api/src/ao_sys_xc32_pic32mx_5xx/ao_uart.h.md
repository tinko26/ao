# Constants

## `AO_UART_BRG_BITS`

```c
#define AO_UART_BRG_BITS (16)
```

The size of the `UxBRG.BRG` field, in bits.

## `AO_UART_FIFO_CAPACITY`

```c
#define AO_UART_FIFO_CAPACITY (8)
```

The capacity of the receive FIFO, in bytes.
