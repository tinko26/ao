# Notes

This module defines a single macro constant indicating whether an application is running in debug mode.

# Include

```c
#include <stdbool.h>
```

# Constants

## `AO_DEBUG`

Indicates whether the application is running in debug mode.

```c
#define AO_DEBUG
```
